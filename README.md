# Axilock Blog

[![Blog Deploy](https://github.com/axilock/blog/actions/workflows/jekyll.yml/badge.svg)](https://github.com/axilock/blog/actions/workflows/jekyll.yml)

## Add new post

1. Add new file at ``_posts/YYYY-MM-DD-NAME.md``
1. You can refer to assets at ``assets/``

## Run locally

1. Run ``bundle exec jekyll serve -H 0.0.0.0``

## Deploy

```
git add . && git commit -m "Blog on $DATE" && git push
```
